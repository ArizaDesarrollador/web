Motivación personal.

¿Por qué he optado por diseñar una web estática?


Como he comentado en uno de los motivos del artículo anterior, una página estática hay que construirla desde 0 y esto conlleva tener conocimientos de programación para poder desarrollarla, pero a su vez de esta manera mejoras en el ámbito del desarrollo.

Por el contrario un Gestor de contenido (CMS) como WordPress, no es necesario tener conocimientos de programación para poder administrar tu página web. Pero hay que tener claro, que si alguien es muy bueno en este software puede crear páginas webs muy profesionales como también si no tienes unos conocimientos no podrás obtener los resultados deseados. Todo lleva su tiempo, y la curva de aprendizaje dependerá del usuario.

Por ello hay que tener claro dos conceptos:


Si quieres una página web que no necesita almacenar nada en una base de datos y quieres una página web rápida y totalmente personalizable, entonces desarrolla una página web estática.


Pero si necesitas usar una base de datos para poder guardar datos de los usuarios en tiendas o comercios online y crear una página web rápida, entonces deberás optar por un CMS, ya que imaginaros crear una tienda de ropa y tener que hacer una página estática para cada artículo, sería una auténtica locura.
